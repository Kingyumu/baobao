# 桌面天气站完整制作指南

> 从零到成品的制作手册：备齐材料 → 接线 → 编译烧录固件 → 装外壳 → 配网送礼。  
> 本文档**不包含项目源代码**，固件逻辑见仓库 `src/` 目录及源码注释。

**主控：** Raspberry Pi Pico 2 W（RP2350，板载 WiFi）  
**传感器：** BME280（温湿度气压）+ DS3231（RTC）  
**显示：** 3.5 寸 ILI9488 彩色 TFT（480×320）  
**交互：** 无源蜂鸣器 + TTP223 触摸

## 1. 制作流程一览

按顺序完成以下步骤，即可得到可送礼的成品：


| 步骤    | 做什么                      | 预计时间       | 完成标志                    |
| ----- | ------------------------ | ---------- | ----------------------- |
| ① 采购  | 按材料清单买齐模块与工具             | 1–3 天（等快递） | 所有模块到手                  |
| ② 搭环境 | 安装 Rust、交叉编译工具链、probe-rs | 30–60 分钟   | `cargo`、`probe-rs` 命令可用 |
| ③ 接线  | 面包板或洞洞板连接各模块             | 30–60 分钟   | 对照检查清单全部打勾              |
| ④ 烧录  | 克隆项目、编译、写入 Pico 2 W      | 15–30 分钟   | 屏幕出现启动动画                |
| ⑤ 配网  | 手机连热点、网页填家里 WiFi         | 5 分钟       | 连上 WiFi，显示室外天气          |
| ⑥ 外壳  | 3D 打印或手工盒体，固定模块          | 1–2 小时     | 线材不 loose，屏幕对齐窗口        |
| ⑦ 送礼  | 打印附录 B 说明书，装盒            | 15 分钟      | 对方插电即用                  |


**推荐顺序：** 先面包板裸机调通（③④⑤），确认功能正常后再做外壳（⑥），避免返工。

## 2. 成品功能说明

### 2.1 功能一览


| 功能           | 说明                          | 是否需要 WiFi     |
| ------------ | --------------------------- | ------------- |
| 室内温湿度 + 气压   | BME280 实时测量                 | 否             |
| 精确时钟         | DS3231，断电靠纽扣电池走时            | 否（联网可 NTP 校准） |
| 圆形时钟 + 双页 UI | 轻触切换本地页 / 对方城市页；两页均保留时钟圆盘 | 否             |
| 双城室外天气       | 本地城市 + 对方城市（编译前配置城市码）           | 是             |
| 表情动画         | 随温度、天气、气压趋势变化               | 部分依赖联网天气      |
| 室外天气         | 中国天气网，本地 + 对方两城数据          | 是             |
| 纪念日 / 生日     | 到日弹祝福 + 旋律                  | 否             |
| 闹钟 / 整点报时    | 软件检测时间后蜂鸣                   | 否             |
| 气压趋势与变天预警    | 本地气压趋势（变天预警仅本地页显示）         | 否             |
| 触摸互动         | 短触切页；长按 2 秒爱心；长按 5 秒清除 WiFi | 否             |
| 手机配网         | 热点 + 网页，无需改代码烧录             | 首次或换 WiFi 时   |
| BLE 传感器广播    | 手机可连读室内温湿度（联网成功后开启）         | 否（蓝牙独立）       |

rustc --version

## 5. 硬件接线

### 5.1 原则

- **共地：** 所有模块 GND 接 Pico GND  
- **I2C 共享：** BME280 与 DS3231 共用 GP0(SDA)、GP1(SCL)，靠不同地址区分  
- **WiFi：** Pico 2 W 板载，**无需接线**  
- **屏幕 SPI：** 独立一组线，不要与 I2C 混接

### 5.2 接线总览图

```
                    Raspberry Pi Pico 2 W
                    ┌────────────────────────────────────┐
  3V3 (Pin36) ──┬──┼──────┬──────┬──────────┬──────────┼── ...
  GND (Pin38) ──┼──┼──────┼──────┼──────────┼──────────┼── ...
  GP0 (Pin1)  ──┼──┼──────┼──────┼──────────┼──────────┼── ...
  GP1 (Pin2)  ──┼──┼──────┼──────┼──────────┼──────────┼── ...
  GP6 (Pin9)  ──────────────────────────────────────────┼── ...
  GP7 (Pin10) ──────────────────────────────────────────┼── ...
  GP8~GP11    ──────────────────────────────────────────┼── SPI → TFT
                    └────────────────────────────────────┘
         │           │          │              │
    BME280+DS3231  蜂鸣器+触摸   ILI9488 TFT    WiFi 板载
```

### 5.3 引脚对照表


| Pico 引脚      | 功能       | 连接                             |
| ------------ | -------- | ------------------------------ |
| GP0 (Pin1)   | I2C SDA  | BME280 SDA、DS3231 SDA          |
| GP1 (Pin2)   | I2C SCL  | BME280 SCL、DS3231 SCL          |
| GP6 (Pin9)   | GPIO 出   | 蜂鸣器 I/O                        |
| GP7 (Pin10)  | GPIO 入   | 触摸 I/O                         |
| GP8 (Pin11)  | GPIO 出   | ILI9488 DC                     |
| GP9 (Pin12)  | GPIO 出   | ILI9488 CS                     |
| GP10 (Pin14) | SPI SCK  | ILI9488 SCK                    |
| GP11 (Pin15) | SPI MOSI | ILI9488 MOSI                   |
| 3V3 (Pin36)  | 电源       | BME280、DS3231、触摸、TFT RST/LED 等 |
| 5V (Pin40)   | 电源       | 蜂鸣器 VCC（声音更大）；部分 TFT 也可用 5V    |
| GND (Pin38)  | 地        | 所有模块 GND                       |


### 5.4 各模块接线要点

**BME280：** VCC、GND、SDA、SCL 四线即可；确认芯片丝印为 BME280；I2C 地址一般为 **0x76**。

**DS3231：** 同上四线；装好 **CR2032** 纽扣电池；地址 **0x68**。

**ILI9488：** SCK、MOSI、CS、DC、VCC、GND；RST 与 LED 可接 3V3 常亮；MISO 可不接。模块丝印顺序可能不同，**以模块标注为准**。

**无源蜂鸣器：** I/O → GP6；VCC 建议 **5V**；GND 共地。

**TTP223：** I/O → GP7；触摸时输出高电平。

### 5.5 接线检查清单（烧录前必做）

- BME280、DS3231 的 SDA→GP0，SCL→GP1  
- ILI9488：SCK→GP10，MOSI→GP11，CS→GP9，DC→GP8  
- ILI9488 背光 LED、RST 接 3V3（或按模块说明）  
- 蜂鸣器 I/O→GP6  
- 触摸 I/O→GP7  
- 所有 GND 共地  
- DS3231 纽扣电池已装  
- USB 线为**数据线**

## 6. 固件说明（项目能做什么）

仓库里的固件是预写好的程序，你**不需要改代码也能做出成品**（配网用手机完成）。下面说明固件行为，方便排查问题。

### 6.1 项目里有哪些模块（仅文件名，方便查注释）


| 文件                         | 作用                           |
| -------------------------- | ---------------------------- |
| `main.rs`                  | 启动流程、主循环、触摸逻辑                |
| `config.rs`                | 城市码、纪念日、闹钟等可调参数              |
| `sensors.rs`               | BME280、DS3231 驱动             |
| `display.rs` / `render.rs` | 屏幕驱动与界面绘制                    |
| `network.rs`               | WiFi、NTP、网络天气                |
| `wifi_store.rs`            | Flash 保存最多 5 个 WiFi（最近使用的优先） |
| `provisioning.rs`          | 热点 + 网页配网                    |
| `state.rs`                 | 气压趋势、闹钟、纪念日等业务状态             |
| `buzzer.rs`                | 提示音与旋律                       |
| `ble.rs`                   | 蓝牙广播室内传感器（联网成功后启动）           |


详细 Rust 学习注释见各文件顶部说明。

### 6.2 上电后发生什么

1. 屏幕显示 Loading 动画与「天气站启动中…」
2. 读取 Flash 里记住的 WiFi，**按最近使用顺序**逐个尝试连接
3. **若全部连不上** → 进入配网模式（见 7.3），此时**不开启 BLE**
4. **若连上** → 启动 BLE、NTP 对时、拉取天气 → 双声蜂鸣表示就绪
5. 进入主循环：每秒刷新显示，定期更新天气与 NTP

### 6.3 配网模式（送礼核心）


| 项目    | 内容                                      |
| ----- | --------------------------------------- |
| 热点名称  | `Baobao`（开放，无密码）                        |
| 设备 IP | `192.168.4.1`                           |
| 操作    | 手机连热点 → 浏览器打开上述 IP → 填家里 WiFi 名和密码 → 提交 |
| 成功    | 设备验证能连上 → 写入 Flash → 自动重启 → 正常模式        |
| 记忆    | 最多记住 **5 个** WiFi；满员时淘汰**最久未用**的网络      |
| 重新配网  | 长按触摸 **5 秒** → 清除全部 WiFi → 重启进入配网       |


屏幕配网页也会提示上述步骤。

### 6.4 触摸操作


| 操作      | 效果                        |
| ------- | ------------------------- |
| 短按      | 切换页面：本地 ↔ 对方城市（两页均保留时钟圆盘） |
| 长按约 2 秒 | 爱心动画 + 短促提示音              |
| 长按约 5 秒 | 两声长「嘀」→ 清除 WiFi 记忆 → 重启配网 |


### 6.5 网络相关

- **WiFi：** 仅 2.4GHz  
- **室外天气：** 中国天气网；烧录前在 `config.rs` 配置**本地城市码**与**对方城市码**（及显示名称）  
- **NTP：** 默认国家授时中心，联网后自动校准 DS3231  
- **断网：** 室内温湿度、时钟、表情、闹钟仍可用；室外天气不更新

### 6.6 表情与气压

- 表情随温度、网络天气、**气压趋势**变化  
- 气压持续下降且偏低时可能触发**变天预警**（屏幕提示 + 蜂鸣）  
- 不联网也能根据本地气压趋势显示「转晴 / 转阴」类提示

## 8. 外壳制作与组装

### 8.1 建议尺寸

- 外廓约 **100 × 80 × 40 mm**（可按 TFT 实际大小调整）  
- 屏幕开窗约 **78 × 54 mm**（比 3.5 寸可视区略大 1–2 mm）  
- BME280 处留 **φ6 mm 透气孔**（气压测量需要空气流通）  
- 后面留 **USB Type-C 口**

### 8.2 方案选择


| 方案        | 难度  | 效果           |
| --------- | --- | ------------ |
| 3D 打印 PLA | ★★☆ | 推荐，可定制触摸区与开孔 |
| 亚克力盒      | ★☆☆ | 快速，内部可见      |
| 纸盒 / 木盒   | ★☆☆ | 手工温馨感        |


无 3D 打印机可使用嘉立创 3D 打印、淘宝代打，约 ¥10–30。

### 8.3 组装顺序

1. 去支撑、打磨外壳
2. 用热熔胶或螺丝将 **Pico** 固定在底座（USB 口对准后壳开孔）
3. 固定 **TFT**，屏幕对准前窗
4. **BME280** 对准透气孔（勿密封）
5. 固定 **DS3231**、蜂鸣器、触摸模块（触摸金属面朝向壳外指定位置）
6. 理线、热熔胶加固，避免短路
7. 合盖，底脚贴防滑垫

### 8.4 送礼包装建议

- 打印 [附录 B](#附录-b随礼使用说明书) 放入盒内  
- 附 USB 线、备用 CR2032  
- 盒内留简短卡片即可；**无需**教对方改代码或烧录

## 10. 常见问题排查

### 10.1 编译 / 烧录


| 现象                         | 处理                                                          |
| -------------------------- | ----------------------------------------------------------- |
| `can't find crate for std` | 确认用 `cargo build --release`，目标为 `thumbv8m.main-none-eabihf` |
| `No connected probes`      | 检查 USB 数据线、BOOTSEL、Windows 下 Zadig WinUSB                   |
| 烧录成功无显示                    | `probe-rs attach` 看是否 panic；查 SPI 与背光接线                     |


### 10.2 显示


| 现象      | 处理                                 |
| ------- | ---------------------------------- |
| 不亮      | 查 TFT VCC、LED 背光是否接 3V3/5V         |
| 白屏 / 花屏 | 查 SCK/MOSI/CS/DC；确认驱动为 **ILI9488** |
| 颜色不对    | 查 DC 线是否接错                         |


### 10.3 传感器 / 时钟


| 现象        | 处理                                       |
| --------- | ---------------------------------------- |
| BME280 失败 | 查 I2C 线；确认 BME280 非 BMP280；试地址 0x76/0x77 |
| 读数偏高      | 传感器远离 Pico 发热区                           |
| 时间不准      | 联网 NTP 校准；或 DS3231 换 CR2032              |


### 10.4 WiFi / 配网


| 现象        | 处理                                   |
| --------- | ------------------------------------ |
| 一直配网界面    | 正常：Flash 无有效 WiFi；按 6.3 配网           |
| 配网提交失败    | 确认 2.4GHz、密码正确；路由器勿开「AP 隔离」          |
| 换路由器      | 再配一次新 WiFi 即可，旧 WiFi 仍保留在记忆里（最多 5 个） |
| 彻底重置 WiFi | 长按触摸 5 秒                             |


### 10.5 触摸 / 蜂鸣


| 现象       | 处理                       |
| -------- | ------------------------ |
| 触摸无反应    | 查 GP7；用指腹大面积触摸金属面        |
| 只有「嘀」无旋律 | 确认是**无源**蜂鸣器；有源蜂鸣器无法播生日歌 |

#### 📦 它能做什么

- 显示室内**温度、湿度、气压**  
- 显示**时间**（断电后时钟电池仍走时）  
- 连接 WiFi 后显示**本地**与**对方城市**室外天气（送礼前已预设城市）  
- 会根据天气和温度变换**小表情**  
- 特殊日子会**播放旋律**  
- **轻触**顶部金属面可以互动

---

#### 🔌 第一次使用（约 5 分钟）

**1. 插电**  
用 USB 线连接充电器或电脑，屏幕会亮起。

**2. 连接 WiFi（首次必做）**  
若屏幕提示「WiFi 配网」：

1. 手机打开 WiFi，连接 `**Baobao`**（无密码）
2. 浏览器打开 **192.168.4.1**
3. 填写家里 **WiFi 名称**和**密码**（请选 **2.4G** 网络，不要用 5G 专用名）
4. 点保存，等待设备重启

成功后右上角 WiFi 图标变绿，会显示室外温度。

**3. 日常使用**  
之后每次插电会自动连上记住的 WiFi，**不用再配**。  
若搬新家或换了路由器，同样步骤再配一次即可。

**4. 重新配网 / 换 WiFi**  
长按触摸区 **约 5 秒**（两声长「嘀」），设备会清除 WiFi 并重启到配网界面。
