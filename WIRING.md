# 桌面天气站接线指南

## 硬件清单

| 模块 | 型号 | 数量 | 供电 |
|------|------|------|------|
| 主控 | Raspberry Pi Pico 2 W | 1 | USB 5V |
| 温湿度气压传感器 | BME280 模块 | 1 | 3.3V |
| 实时时钟 | DS3231 模块 | 1 | 3.3V |
| TFT 彩屏 | ILI9488 3.5寸 480×320 | 1 | 5V (模块自带稳压) |
| 无源蜂鸣器 | 无源模块（板载 S9012 三极管） | 1 | 3.3V–5V |

```
          ┌─────────────────────┐
      GP0 │ 1              40│ VBUS (5V)
      GP1 │ 2              39│ VSYS
        — │ 3              38│ GND ←── 公共地
        — │ 4              37│ 3V3_EN
        — │ 5              36│ 3V3_OUT ←── 传感器供电
      GP6 │ 6  (蜂鸣器)  35│ ADC_VREF

     GP16 │ 16           25│ GP18

        — │ 18           23│ GP16
        — │ 19           22│ GND

          └─────────────────────┘
```

## 详细接线

### 1. BME280（I2C 传感器）

| BME280 模块 | Pico 2 W | 说明 |
|-------------|----------|------|
| VCC | 3V3_OUT (Pin 36) | **必须 3.3V，接 5V 会烧** |
| GND | GND (Pin 38) | |
| SDA | GP1 (Pin 2) | I2C0 SDA |
| SCL | GP0 (Pin 1) | I2C0 SCL |

> I2C 地址：SDO 接 GND 为 `0x76`（代码默认），接 VCC 为 `0x77`。

### 2. DS3231（RTC 实时时钟）

| DS3231 模块 | Pico 2 W | 说明 |
|-------------|----------|------|
| VCC | 3V3_OUT (Pin 36) | **必须 3.3V** |
| GND | GND (Pin 38) | |
| SDA | GP1 (Pin 2) | 与 BME280 共用 I2C 总线 |
| SCL | GP0 (Pin 1) | 与 BME280 共用 I2C 总线 |

> DS3231 模块自带纽扣电池，断电后时间不丢失。
> I2C 地址：`0x68`（固定，不会与 BME280 冲突）。

### 3. ILI9488 TFT 彩屏（SPI）

| ILI9488 模块 | Pico 2 W | 说明 |
|--------------|----------|------|
| VCC | VBUS (Pin 40) | 5V，模块自带 3.3V 稳压 |
| GND | GND (Pin 38) | |
| SCL / SCLK | GP10 (Pin 10) | SPI1 时钟 |
| SDI / MOSI | GP11 (Pin 11) | SPI1 数据 |
| CS | GP9 (Pin 9) | 片选（软件控制） |
| DC / RS | GP8 (Pin 8) | 数据/命令选择 |
| RESET | 3V3 或悬空 | 复位引脚，接高或悬空 |
| LED / BL | VBUS 或独立控制 | 背光，直接接 5V 常亮 |
| MISO | GP12 (Pin 12) | **不接**（屏只写不读） |

> SPI1 代码中初始化了 MISO (GP12)，但实际未使用，屏的 MISO 可以不接。

### 4. 无源蜂鸣器模块（板载三极管，直连 GPIO）

模块**自带 S9012 三极管 + 限流电阻**，GPIO 直接驱动，无需外接元件。

| 蜂鸣器模块 | Pico 2 W | 说明 |
|-----------|----------|------|
| VCC | 3V3 (Pin 36) 或 VBUS (Pin 40) | 3.3V–5V 均可 |
| GND | GND (Pin 38) | |
| I/O | GP6 (Pin 6) | 高电平触发，方波驱动发声 |

### 5. 触摸按钮

| 触摸模块 | Pico 2 W | 说明 |
|----------|----------|------|
| VCC | 3V3_OUT (Pin 36) | |
| GND | GND (Pin 38) | |
| SIG / OUT | GP7 (Pin 7) | 触摸信号输出 |

> 代码中配置了内部下拉电阻 (`Pull::Down`)，触摸时输出高电平。

## 接线顺序（安全操作）

```
第 1 步：断开 USB，确认 Pico 未通电
    ↓
第 2 步：接 GND 线 — 所有模块的 GND 先连到 Pico 的 GND
    ↓
第 3 步：接电源线 — 3V3_OUT 给传感器，VBUS 给屏幕
    ↓
第 4 步：接信号线 — I2C / SPI / GPIO
    ↓
第 5 步：目视检查 — 有无短路、虚焊、错接
    ↓
第 6 步：万用表测 VCC 与 GND 是否短路
    ↓
第 7 步：插 USB 上电
```

## 常见烧毁场景与预防

| 场景 | 后果 | 预防 |
|------|------|------|
| BME280/DS3231 接 5V | 芯片烧毁 | 确认接 3V3_OUT，不是 VBUS |
| I2C 接线时热插拔 | IO 闩锁烧毁 | 断电后再改线 |
| SPI 屏 VCC 接 3.3V | 可能不亮（背光电流大） | 屏幕接 VBUS (5V) |
| GND 没共地 | 通信不稳定或损坏 | 所有模块 GND 必须连在一起 |

## 检查清单

- [ ] 所有模块 GND 已共地
- [ ] BME280 和 DS3231 接 3.3V，不是 5V
- [ ] ILI9488 接 5V (VBUS)
- [ ] I2C 只有两根线（SDA、SCL），没有接反
- [ ] 蜂鸣器模块 I/O 接 GP6
- [ ] 触摸模块信号线接 GP7
- [ ] 无裸露导线短路风险
- [ ] 万用表测量 VCC-GND 无短路
